How to Find Your Vauxhall Serial Number

1. How to show your Vauxhall Connect radios information on the radios display.
  • Turn the ignition switch ON.
  • it will show 'Code:'.
  • To get the information we need to decode your radio, you will need to enter an incorrect code in 3x times. For example, enter: 1-2-3-4.
2. What To Do Next
  • The 'Serial number', 'Device number' and 'Date'. should display on the radios display. (Please see image for reference).

Method 2 - Vauxhall Radio Code By The Label
How to find your Vauxhall radio serial on the radios label.
  • To find your Vauxhall serial number, you will need to remove the radio using radio release keys. Once the radio has been removed you will need to locate the radios label, which is on the top or side of the radios casing.

How to Enter Vauxhall Connect Bosch Radio Code

  • When the radio is turned on 'CODE IN' should display on the radios display to input:
    • Repeatedly press button 1 until the correct first digit of the code is displayed.
    • Repeatedly press button 2 until the correct second digit of the code is displayed.
    • Repeatedly press button 3 until the correct third digit of the code is displayed.
    • Repeatedly press button 4 until the correct fourth digit of the code is displayed.
  • If you have entered the code correctly on the radios display, simply press enter.

How to Enter your Vauxhall Radio Code

Using The Buttons On The Unit

  1. Switch on the CD or cassette player.
  2. The display should show 0 0 0 0.
  3. Press 1st then 2, 3 and 4 repeatedly until each digit of your code appears on the display
  4. Press and hold 6 to activate the unit.

Using The Stalk Control

  1. Switch on the CD or cassette player.
  2. The display should show 0 0 0 0.
  3. Rotate the thumbwheel for the 1st digit of the code.
  4. Then press the bottom button to move on.
  5. Rotate the thumbwheel for the 2nd digit of the code.
  6. Then press the bottom button to move on.
  7. Rotate the thumbwheel for the 3rd digit of the code.
  8. Then press the bottom button to move on.
  9. Rotate the thumbwheel for the 4th digit of the code.
  10. Then press & hold the bottom button button to activate.
