How To Get My Chrysler Radio Code

Removing Your Radio

It is easy to remove a radio. You need a flat-head screwdriver & a prying tool to remove the stereo.

Model variants require release keys to remove your radio from the dashboard.

Using the release keys, push them into each corner of the radio, apply pressure and the radio will release from the dashboard. The radios label is located on the radio's casing.

Get Your Serial Number

Once removed, you need to locate the radios label.

Chrysler serial numbers begin with one of the following:

  • TQ
  • T00
  • TC
  • TD
  • TZ
  • TB
  • TH
  • TM9
  • TVP
  • T16
  • T19
  • T0MY
  • T82

How to Enter Your Chrysler Radio Code

Chrysler Radio Code Input

  1. Turn the radio on.
  2. 'CODE' will display on the screen at this point.
  3. Press the peset button 1 multiple times until the first digit is on-screen.
  4. Repeat with buttons 2, 3, 4 until the correct code is displayed.
  5. To confirm the code, Chryslers have a “Enter” or “OK” button. Hold down for 2 – 5 seconds.
  6. If your radio doesn't come on, the code may be incorrect, contact us for futher guidance.

Other Chrysler Entry Methods

  • Turn on your ignition so the radio is on.
  • Next the code must be typed using the preset buttons.
  • The code will only have buttons 1 - 6 included as this is the only pre-set buttons on the unit
  • When the last number of the decode is entered, the radio will come on instantly.
