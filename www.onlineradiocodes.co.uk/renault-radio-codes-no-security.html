How to Enter your Renault Radio Code

Using The Buttons On The Unit
  1. Switch on the CD or cassette player.
  2. The display should show 0 0 0 0.
  3. Press 1st then 2, 3 and 4 repeatedly until each digit of your code appears on the display
  4. Press and hold 6 to activate the unit.
Using The Stalk Control
  1. Switch on the CD or cassette player.
  2. The display should show 0 0 0 0.
  3. Rotate the thumbwheel for the 1st digit of the code.
  4. Then press the bottom button to move on.
  5. Rotate the thumbwheel for the 2nd digit of the code.
  6. Then press the bottom button to move on.
  7. Rotate the thumbwheel for the 3rd digit of the code.
  8. Then press the bottom button to move on.
  9. Rotate the thumbwheel for the 4th digit of the code.
  10. Then press & hold the bottom button button to activate.
