How To Find Your Mercedes Radio Code Serial
1. How To Take Out An Alpine Radio

It is quite simple to remove your Alpine radio. You will need a set of radio release keys (PC5-132). These are inexpensive and are around Â£3 - Â£5, you can obtain them from a local car audio shop or Online.

Once you have the radio release keys, push them into the 2 holes in each bottom corner of the radio (See image), apply a little pressure and the radio will release from the dashboard. The radios label is located on the radio's casing.

2. What To Do Next

The Alpine radio serial starts with AL and is 14 digits long. For example AL 2910 Y 01 04000. This is usually on the back of the radio, engraved underneath a white label. Please see image for reference.

How to Enter your Mercedes Alpine Radio Code

With Buttons 1 - 4 - 5 digit preset
  • When the radio is turned on 'CODE' should display on the radios display.
  • If the power supply to the radio has been interrupted, "CODE" will appear on the display when it is next switched on.
    • Press/hold button 1 until the correct 1st digit of the code displays.
    • Press/hold button 2 until the correct 2nd digit of the code displays.
    • Press/hold button 3 until the correct 3rd digit of the code displays.
    • Press/hold button 4 until the correct 4th digit of the code displays.
    • Press/hold button 5 until the correct 5th digit of the code displays.
  • If you have entered the code correctly on the radios display, simply press and hold one of the arrow buttons to confirm your entry.
With Stalk Control - 5 digit preset
  • When the radio is turned on 'CODE' should display on the radios display.
  • Using the right-hand control knob, enter each number of the radio code and confirm this by pressing the control knob each time.
  • If you have entered the code correctly on the radios display, push and hold right control knob for about 3 seconds.
Helpful Information
  • If an incorrect numerical code has been entered and confirmed, "CODE" will reappear on the display. The correct code must be entered once again. If an incorrect code is entered three times, "WAIT" will appear on the display and the radio will be isolated for about 10 minutes.
